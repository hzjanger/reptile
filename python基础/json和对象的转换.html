# json和对象的转换

可以调用JSON库中的loads方法将JSON文本字符串转为JSON对象,可以通过dumps方法将JSON对象转为文本字符串

# json字符串转对象

使用json的loads方法

import json

string = '''
[
{"name": "bob", "gender": "male", "birthday": "1992-10-18"},
{"name": "Selina", "gender": "female", "birthday": "1995-10-18 "}
]
'''

print(type(string))
data = json.loads(string)
print(type(data))
print(data)

TIP

需要注意的是不能使用单引号
